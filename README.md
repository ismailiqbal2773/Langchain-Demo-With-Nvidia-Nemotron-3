**Langchain-Demo-With-Nvidia-Nemotron-3**
A sleek AI Chatbot built using the LangChain framework and powered by the NVIDIA Nemotron-3 (120B) model. This project demonstrates how to integrate high-parameter LLMs via OpenRouter into a Streamlit web application.

**Live Demo**
Experience the chatbot live here:
https://langchain-demo-with-nvidia-nemotron-3-fgniaxq4bdme5ywwcfvqs8.streamlit.app/

**Key Features**
Advanced LLM: Utilizes NVIDIA Nemotron-3 Super (120B) for high-quality responses.

**Orchestration**: Built with LangChain (Prompt Templates & Output Parsers).

**Responsive UI**: Minimalist and fast interface powered by Streamlit.

**Secure Architecture**: Uses environment variables and Streamlit Secrets to protect API keys.

**Tech Stack**
**Language**: Python 3.x

**Framework**: LangChain

**UI**: Streamlit

**API Provider**: OpenRouter

**Model**: nvidia/nemotron-3-super-120b-a12b:free

**Quick Start (Local)**
**Clone the Repo:**
git clone https://github.com/ismailiqbal2773/Langchain-Demo-With-Nvidia-Nemotron-3.git

**Install Dependencies**:
pip install -r requirements.txt

**Environment Setup**:
Create a .env file and add your key:
OpenRouter_API_Key=your_api_key_here

**Run the App**:
streamlit run NVIDIA_CHATBOT.py
